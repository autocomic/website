## Contributing

1.  **Install the Gatsby CLI**

    ```shell
    npm install -g gatsby-cli
    ```

2.  **Install dependencies**

    ```shell
    npm install
    ```

3.  **Start developing**

    ```shell
    gatsby develop
    ```

4. **Test your changes locally**

    ```shell
    gatsby build
    gatsby serve
    ```

4. **Deploy to GitHub Pages**

    ```shell
    npm run deploy
    ```
